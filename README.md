# Teacher Maker

교사를 위한 30분 바이브 코딩 입문 웹앱입니다.

## 목적

개발을 몰라도 AI와 함께 첫 번째 작은 앱을 만들어 보는 경험을 제공합니다.

## 주요 기능

- 30분 로드맵
- 실습 프로젝트 2개
- 프롬프트 라이브러리
- FAQ
- 진행 체크와 브라우저 저장
- 복사하기 버튼

## 실행 방법

```bash
npm install
npm run dev
```

## 배포

Vercel 배포를 권장합니다.

## 문서

PRD 기준으로 MVP 범위만 구현했습니다. 로그인, 회원가입, 서버 저장, 관리자 페이지, 실제 AI 연동 기능은 포함하지 않습니다.
